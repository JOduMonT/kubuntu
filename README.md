# Kubuntu

Nuts and Bolts to configure KDE Ubuntu Desktop

### What is the goal of that REPO
1. Make my Ubuntu Desktop stable but also faster than the stock version
2. Having all my tools and favorite softwares/configuration once I deployed it on a new machine (laptop/VM/...)
3. Help you to enjoy Ubuntu and make KDE your favorite GUI
