# Ubuntu

Nuts and Bolts to configure Ubuntu Desktop

### What is the goal of that REPO
1. Make my Ubuntu Desktop stable but also faster than the stock version
2. Having all my tools and favorite softwares/configuration once I deployed it on a new machine (laptop/VM/...)
3. Help you to enjoy Ubuntu

### What you have the right to do
- **Issues**: Submit questions or issues
- **Fork**: Make you own copy and go solo
- **PR accepted**: Propose your improvements because together we do better

### Disclaimer

These scripts are tested on Ubuntu LTS only and with KDE.
You are free to review, use, fork and propose change.
